﻿Các chức năng chưa hoàn thiện:
 1. Vẽ:
    + Cung tròn
    + Cung Ellipse
 2. Đổi màu pen, brush
 3. Xoay hình
 4. Save
 5. Một số chức năng nâng cao
Các chức năng đã làm:
 1. Vẽ: Đường thẳng, tròn, ellipse, chữ nhật, hình bình hành, parabola, hyperbola
   polylines
 2. Di chuyển
 3. Resize
Sử dụng:
  Có 2 chế độ: 
    + mode = 0: Chế độ vẽ
    + mode = 1: Chế độ default: để chọn, move, scale
=> Click vào icon cursor để thực hiện mode = 1.
